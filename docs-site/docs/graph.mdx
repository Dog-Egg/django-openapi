# 流程图

### Schema 反序列流程

```mermaid
graph TD
    input(input)
    isnull{is null?}
    nullable{nullable?}
    error(raise DeserializationError)
    validate[validations]
    deserialize[_deserialize]
    output(output)

    input --> isnull
    isnull --> |yes|nullable
    isnull --> |no|deserialize
    nullable --> |yes|output
    nullable --> |no|error
    deserialize --> |ok|validate
    deserialize --> |error|error
    validate --> |ok|output
    validate --> |error|error
```

### Schema 序列流程

```mermaid
graph TD
    input(input)
    isnull{is null?}
    nullable{nullable?}
    error[raise SerializationError]
    serialize[_serialize]
    output(output)
    fallback[fallback]

    input --> isnull
    isnull --> |yes|nullable
    isnull --> |no|serialize
    nullable --> |yes|output
    nullable --> |no|error
    serialize --> |ok|output
    serialize --> |error|error
    error -.-> fallback
    fallback -.-> output
```
