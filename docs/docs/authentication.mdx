## 权限组合

**permission 类/对象可使用 `&` 和 `|` 运算符来实现 `and` 和 `or` 的操作。**

以下示例将获得一个新的 permission 对象，表示用户需要是管理员或拥有 permission_1 权限。

``` python
permission = IsAdministrator | HasPermission('permission_1')
```
